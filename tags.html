Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header container for intro content
div division defines a division or section
ol ordered list creates an ordered list
ul unordered list creates an unordered list
li list adds list items
blockquote blockquote section that is quoted from another source
!DOCTYPE doctype HTML declaration
br line break inserts a single line break
header header container for introductory content

Inline Elements

img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code make text look like code <code>technical sophistication<code> technical sophistication
span span container used to mark up a part of text <span>technical sophistication<span> technical sophistication
!-- comment tag make comments that are not displayed in browser <!-- ... --> <!--technical sophistication-->

Document Tags

Tag Name Purpose
html HTML container for all other HTML elements
head head container for metadata
body body defines documents body
title title defines the document title
meta meta defines metadata of a document
